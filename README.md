# 🎥 Movie Genre Classification 

## 📌 Project Description
This project is a Machine Learning-based system that classifies movie genres based on their textual descriptions. It uses Natural Language Processing(NLP) techniques to analyze the input text and predict the most appropriate genre.

---

## 🎯 Objectives
- To develop a system that automatically classifies movie genres
- To apply machine learning techniques for text classification
- To understand and implement Natural Language Processing concepts
- To improve accuracy through proper data preprocessing

---

## ⚙️ Technologies used
- Python
- Pandas
- Scikit-learn

---

## 📊 Dataset
The dataset contains movie descriptions and their corresponding genres.

## Example:

| Description | Genre |
|-------------| ------|
| A hero saves the world from enemies | Action |
| A funny story of friends on a trip | Comedy |

----

## 🔄 Methodology
The project follows these steps:

1. Data Collection
2. Data Preprocessing
3. Feature Extraction using CountVectorizer
4. Model Training using Naive Bayes
5. Prediction of movie genre

----

## 💻 Implementation Steps
- Load dataset using Pandas
- Separate features (description) and labels (genre)
- Convert text into numerical format using CountVectorizer
- Split dataset into training and testing sets
- Train model using Multinomial Naive Bayes
- Predict genre based on user input

----

## ▶️ How to Run

1. Install required libraries:
   pip install pandas scikit-learn
2. Run the program:
   python main.py
3. Enter movie description when prompted

----

## 📈 Output Example

Input:
A soldier fighting enemies in war

Output:
Action
